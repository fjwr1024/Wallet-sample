# sample wallet

Next.js(ver13) solana-suite を用いたウォレットシステムサンプル

## ディレクトリ構成

components のみ

### 環境構築

## 1. `git clone`

```bash
git@github.com:Fujiwara-Ken/Wallet-sample.git
```

## 2. `npm install`

`npm install`

```shell
npm -v

npm install
```

## 3. `npm run dev`

`npm run dev`でローカルサーバーを立ち上げ。

```bash
npm run dev
```
